# Script Agent Configuration

## Input Requirements
- Topic/Brand name
- Channel (David Alex / Vibes Academy / Scent Atlas)
- Offer details (if sponsored)
- Target length (8-12 minutes)
- Angle preference (if any)

## Output Structure

### 1. SEO Titles (5 options)
Format: [Primary Keyword] + [Benefit/Hook] + [Year/Qualifier]

### 2. Hooks (3 options)
Using 3-step formula:
1. Context Lean (relatable setup)
2. Scroll Stop (pattern interrupt)
3. Contrarian Snapback (unexpected take)

### 3. Full Script Sections
- **Hook** (0:00-0:30)
- **Intro** (0:30-1:00) - "What is [Product] at its core?"
- **Problem** (1:00-2:00)
- **Solution Overview** (2:00-3:00)
- **Deep Dive** (3:00-7:00) - Features, walkthrough, examples
- **Pricing** (7:00-8:00) - Before limitations
- **Limitations** (8:00-9:00) - Honest, softer for sponsors
- **Who It's For** (9:00-10:00)
- **CTA** (10:00-11:00) - "Before we dive into this, I've partnered..."
- **Outro** (11:00-12:00)

### 4. Talking Points
- Bullet list for each section
- Key phrases to hit
- Sponsor mention placements

### 5. B-Roll Suggestions
- Screen recordings needed
- Stock footage ideas
- Text overlays

## Voice Guidelines

- Conversational: "Let's talk about..." not "What X actually is"
- Direct, no fluff
- "Surprisingly delightful" for great features
- Honest about weaknesses
- Empowering, not prescriptive

## Quality Checklist

- [ ] 8-12 minute spoken length
- [ ] 5 SEO titles included
- [ ] 3 hooks using 3-step formula
- [ ] Sponsor mention in first 60s (if applicable)
- [ ] Pricing before limitations
- [ ] Honest limitations section
- [ ] Strong CTA with offer
- [ ] Sounds like David Alex
